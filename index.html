During my bootcamp at 4Geeks Academy. I Collaborated with a team to design a movie discovery website that allows visitors to search any movie available and create watch lists for their most anticipated ones after creating their own accounts. Users can also add their already finished movies to the completed list, and are able to remove any movie from their lists.

The ReactJS framework was used alongside HTML, CSS, and Bootstrap to build the web application. We fetched the movie details from an API.
My role was to create and manage user accounts and authentication through Firebase
It also included storing the details of the movies the user selected into their personal list using Firestore.
