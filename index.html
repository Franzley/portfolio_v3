Nyaanime Anime Merch Shop

Nyaanime Anime Merch Shop

Welcome to Nyaanime Shop, your go-to destination for anime merchandise! This website is a project inspired by the original design of the Nekotwo website to show my skills in recreating a site by seeing its appearance and functionality.

The shop is built with a responsive design to enhance user experience across devices. The site adapts to different screen sizes so users can visit the site comfortably whether on desktop, tablet, or mobile phone. This project also shows my experience with the EJS templating language and using the 11ty static site generator.
Each product generates its own webpage with a common layout using markdown files which stores each product's details.
Uses decapcms which allows easy creation of product md files
Through this experience I learned more about responsive layouts, and responsive image resolutions and how they affect load.

YouTube Emulation Application

YouTube Emulation Application

I developed a full-stack web application that emulates Youtube's appearance and features, including the ability to view channels, search for videos and recieve a list of results where they can watch their preferred videos or view related videos.

The project was implemented using Material UI, ReactJS, Rapid API, and YouTube V3 API.
The object data was fetched using Rapid API powered by YouTube V3. This allowed access to video and channel names, IDs, descriptions and more.
The site also contains a dark mode which makes use of their local storage to retain their choice so users can comfortably experience the site in their preferred method.
